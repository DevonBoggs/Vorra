# DevonSYNC — WGU AI Study Planner

A desktop Electron app for WGU students to manage their degree with AI-powered study planning, deep course context, and exam intelligence.

## Features
- AI Tool-Use Integration — Anthropic, OpenAI, or compatible API
- Deep Course Context — 35+ structured fields per course
- Degree Plan Parser — Upload a screenshot, AI extracts all courses
- Smart Planning — Start date/time, target graduation, exception days, safety buffer
- Course Ordering — Drag-drop or numbered priority
- Study Radio — 44 SomaFM stations + 50 YouTube live streams
- Practice Exams — AI-generated MCQ from course context
- Study Timer — Per-task timer visible in sidebar
- Focus Pulse — 15-minute check-ins tracking focus streaks
- Weekly Reports — Detailed study analytics
- 5 Themes — Dark, Light, Warm, Mono, Ocean
- Import/Export — CSV + JSON backup/restore

## Setup (Windows)
1. Install Node.js (v18+): https://nodejs.org
2. Extract this folder
3. Double-click setup.bat (first time)
4. After first run, use start.bat
